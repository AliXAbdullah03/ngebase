# **App Name**: Next Global Express

## Core Features:

- Shipment Creation: Admin enters client, cargo, and schedule details. System generates a unique Tracking ID, stores shipment in database, and sends SMS/Email to client with tracking ID.
- Status Updating: Admin updates status per shipment or bulk updates by departure date. System automatically notifies clients (optional toggle).
- Shipment Status List: Statuses appear in correct order with date/time stamp and optional notes.
- Client Tracking Page: User enters Tracking ID and website shows client information, cargo description, destination, departure schedule, tracking ID, and shipment timeline. Current status is highlighted.
- Admin Panel: Login system, dashboard summary, add new shipment form, search shipments, status update page, bulk update feature, export shipments, and notification logs.
- Auto-generate tracking ID: Automatically generate a unique tracking ID for each shipment.
- Auto SMS or Email Notification: Automatically send SMS or Email notifications when shipment is created or status is updated.
- Tracking Page: Search box for Tracking ID. Show result in professional cargo-tracking layout. Mobile-friendly. Show error message if tracking ID not found.

## Style Guidelines:

- Professional cargo-tracking layout for the tracking page.
- Mobile-friendly design for the tracking page.
- Branding with logo and KN Express style (optional).
