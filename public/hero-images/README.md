# Hero Images Directory

This directory contains the hero slider images for the homepage.

## How to Add Images

1. Place your hero images in this folder (`public/hero-images/`)
2. Go to the Admin Panel → Web Page Management → Hero Images
3. Click "Add Image" button
4. Enter the image path (e.g., `/hero-images/your-image.jpg`)
5. Enter an optional title for the image
6. Click "Add Image"

## Image Requirements

- Recommended size: 1200x800px or larger
- Format: JPG, PNG, or WebP
- File names should be descriptive (e.g., `hero-logistics-1.jpg`)

## Example

If you place an image at `public/hero-images/logistics-hero.jpg`, you would enter:
- Path: `/hero-images/logistics-hero.jpg`
- Title: `Fast. Reliable. Secure.`

The images will automatically appear in the hero slider on the homepage.
